# Умный конвертер валют

Веб-сервис для конвертации валют по курсам ЦБ РФ с историей запросов и графиком изменения курса за период. Проект для курса «Облачные технологии» — предназначен для запуска в облаке (Yandex Cloud и др.).

## Возможности

- Конвертация между рублём и основными валютами (USD, EUR, CNY, GBP, JPY, CHF, TRY, KZT)
- Курсы берутся с [ЦБ РФ](https://www.cbr-xml-daily.ru/)
- Краткий тренд: «курс растёт / падает / без изменений» за выбранный период (7/14/30 дней)
- График курса за период (на странице после конвертации)
- История конвертаций сохраняется в SQLite (при деплое в облаке можно заменить на Managed PostgreSQL)

## Локальный запуск

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

2. Запустите приложение:
   ```bash
   python app.py
   ```

3. Откройте в браузере: [http://127.0.0.1:5000](http://127.0.0.1:5000)

## Деплой в Yandex Cloud

### Вариант 1: Виртуальная машина (Compute Cloud)

1. Создайте ВМ с Ubuntu, откройте порт 5000 (или 80 с nginx).
2. Установите Python 3.10+, склонируйте/загрузите проект.
3. Установите зависимости и запустите через gunicorn:
   ```bash
   pip install -r requirements.txt gunicorn
   gunicorn -w 1 -b 0.0.0.0:5000 app:app
   ```
4. Для продакшена настройте systemd или запуск через screen.

### Вариант 2: Serverless Containers

1. Соберите Docker-образ с приложением (Dockerfile: Python, `pip install -r requirements.txt`, `CMD ["gunicorn", "-b", "0.0.0.0:8080", "app:app"]`).
2. Загрузите образ в Container Registry.
3. Создайте сервис в Serverless Containers с этим образом, порт 8080.
4. Выдайте публичный URL сервиса.

### База данных в облаке

- Сейчас используется SQLite (файл `converter.db`). Для облака можно подключить **Yandex Managed Service for PostgreSQL**: задать переменную окружения с connection string и заменить в `app.py` работу с SQLite на `psycopg2` (таблица `history` с теми же полями).

## Технологии

- **Backend:** Python 3, Flask
- **Данные:** API ЦБ РФ (JSON), SQLite
- **Фронт:** HTML, CSS, JS, Chart.js (график)

## Файлы

- `app.py` — основное приложение (маршруты, конвертация, история, тренд)
- `templates/index.html` — страница с формой и историей
- `requirements.txt` — зависимости Python
